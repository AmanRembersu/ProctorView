Introduction

ProctorView is a secure, AI-powered online examination platform with real-time proctoring, identity verification, and intelligent monitoring. This documentation includes setup, architecture, components, APIs, and deployment instructions.

Tech Stack

  • Frontend: React + TailwindCSS + ShadCN UI
  • Backend: Node.js, Express.js
  • Database: PostgreSQL (or SQLite for local)
  • Authentication: JWT
  • Realtime: Socket.IO
  • Proctoring: WebRTC + TensorFlow.js

Folder Structure

/ProctorView
├── client/           # React frontend
│   ├── components/   # UI components (Navbar, Footer, Cards, etc)
│   ├── pages/        # Home, Docs, Dashboard, etc.
│   ├── styles/       # TailwindCSS and global styles
│   └── App.tsx       # Root component
├── server/           # Node.js backend
│   ├── routes/       # Auth, Exam, User
│   ├── controllers/  # Logic handlers
│   ├── models/       # DB models
│   ├── middlewares/  # JWT, rate limiters
│   └── index.js      # Express app
├── README.md
├── .env
└── package.json

Core Features

  • Secure user authentication with JWT
  • Live camera & screen monitoring using WebRTC
  • AI-based face detection and identity validation
  • Realtime alerts for suspicious behavior
  • Admin dashboard to manage exams and results
  • Responsive UI for students and proctors

API Routes

POST   /api/auth/register      → Create user
POST   /api/auth/login         → Login user (JWT)
GET    /api/user/me            → Get current user
POST   /api/exam/start         → Start exam session
POST   /api/exam/submit        → Submit exam
POST   /api/proctor/alert      → Send proctor alert
GET    /api/admin/results      → Admin view results

Deployment

Frontend

cd client
npm install
npm run build
npm run preview

Backend

cd server
npm install
node index.js

Production (Docker)

docker-compose up --build

Conclusion

ProctorView is a scalable, secure, and intelligent platform for academic institutions and corporates to conduct remote assessments. This documentation provides the core details to get started, build, and deploy it efficiently. Contributions are welcome!
